# Yummie_FoodApp_PetProjectApp (в процессе разработки)

Приложение доставки еды с использованием открытого апи ("https://yummie.glitch.me/dish-categories")

Использовались:
- SwiftUI
- Combine
- MVVM
- SDWebImageSwiftUI
- кастомные модификаторы, вью, расширения
